## 🔧 Запуск
Открой `index.html` в браузере (рекомендуется через Live Server).

Сборка не требуется — все зависимости через CDN.

## 💡 Особенности реализации
- **Данные отделены от представления** — тарифы, отзывы и FAQ хранятся в `data.js` и рендерятся через JavaScript
- **Плавный скролл** — `scroll-behavior: smooth` с компенсацией высоты навбара
- **Анимация секций** — fadeInUp при загрузке страницы
- **Hover-эффекты** — карточки приподнимаются, иконки увеличиваются
- **Доступность** — ARIA-атрибуты у карусели и аккордеона
- **Адаптив** — mobile-first, корректное отображение от 320px до 4K

## 🎯 Для портфолио
Этот проект демонстрирует:
- Умение верстать адаптивные лендинги на Bootstrap
- Работу с компонентами (carousel, accordion, navbar, modal)
- Кастомизацию Bootstrap через CSS-переменные
- Организацию кода (данные отдельно от представления)
- Внимание к деталям (анимации, доступность, типографика)
