# MindFlow Landing Page

Адаптивный лендинг для AI-приложения заметок. 

## Стек
- Bootstrap 5.3 (CDN)
- Vanilla JS (ES6+)
- CSS-переменные для кастомной темы
- Архитектура: данные отделены от представления

## Структура
- `index.html` — точка входа
- `style.css` — кастомная фиолетовая тема + анимации
- `js/data.js` — все тексты и данные
- `js/main.js` — рендеринг динамических секций

## Запуск
Открой `index.html` в браузере. Сборка не требуется.
