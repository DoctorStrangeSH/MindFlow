## 📋 О проекте

**MindFlow** — адаптивный лендинг для вымышленного AI-приложения, которое превращает голосовые заметки в структурированные списки задач. Проект демонстрирует навыки вёрстки, дизайна и организации кода.

### 🎯 Что показывает проект
- Чувство дизайна и типографики
- Умение работать с Bootstrap 5.3
- Понимание компонентного подхода
- Навыки кастомизации через CSS Variables
